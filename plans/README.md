# Plans

Технические планы реализации фич. **Один план = одна функция.**

## Имена файлов

- Планы фич: `YYYY-MM-DD-feature-slug.md`
- Планы недели (опционально): `weeks/YYYY-WNN.md` (ISO-неделя)

## Правила

- **Один план = одна функция.** Не «сделать проект», а «добавить экспорт CSV».
- **В каждой фазе — чекбоксы** `[ ]` / `[x]`. После каждой фазы — обновлять.
- **В конце — блок «Итог».** Зеркало предыдущих секций: что реализовано / что не сделано / что в бэклог / уроки.
- **План — снимок состояния**, доступный новому чату. После сессии — актуализировать.
- **Закрытые планы НЕ удаляем.** Архив — это память проекта.

## Создание

Используй skill `/plan <name>` или вручную:

```bash
DATE=$(date +%Y-%m-%d)
cp plans/TEMPLATE.md "plans/${DATE}-my-feature.md"
```
