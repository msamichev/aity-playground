# Prompts Index

Библиотека готовых промптов. **Перед написанием нового — загляни сюда.**

## Категории

| Категория | Путь | Что внутри |
|---|---|---|
| Setup | (пусто) | Промпты для настройки окружения, инструментов |
| Methodology | (пусто) | Методологические приёмы: критика плана, импорт проекта, weekly-planning |
| Domain | (пусто) | Доменные промпты (заполняются по мере появления специфики) |

## Правило

Любой промпт, который используется **больше двух раз** — выноси в этот индекс.
