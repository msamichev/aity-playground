# inbox/

Сюда сваливаешь всё, что собрал перед стартом проекта:

- Экспорты чатов с LLM (md, txt, json)
- Расшифровки голосовых
- Файлы документов (pdf, docx)
- Скриншоты
- Произвольные мысли в любом формате

**Без внутренней структуры.** Не нужно сортировать. `/init-project` сам прочитает и разложит:
- Идейные материалы → `docs/idea/`
- Продуктовая часть → `docs/product/`
- Архитектурные намёки → `docs/architecture/` (на Phase 1)
- Принятые решения → `docs/adr/`

После `/init-project` папка остаётся как **архив**. Не удаляется автоматически.

## Что делать если есть приватные файлы

Положи в `inbox/private/` — эта подпапка в `.gitignore`. `/init-project` всё равно прочитает локально, но не закоммитит.
