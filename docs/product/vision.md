# Vision

> Один абзац: что строим, для кого, какую проблему решает.

`aity-playground` — учебный backend-сервис (todo-list REST API, хранилище в памяти)
для отладки механики контейнерных ботов «aity»: задача → feature-ветка → гейты → MR →
ревью. Ценность не в самом todo-API, а в предсказуемой, легко тестируемой нагрузке, на
которой проверяется автономный цикл ботов. Сервис намеренно минимален и одноразов.
