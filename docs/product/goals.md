# Goals

> Цели на ближайшие 3-6 месяцев. Метрики «получилось».

Горизонт — недели (полигон, не продукт). «Получилось» = на этом репо отлажен цикл ботов.

**Цели:**
- Рабочий baseline todo-API: CRUD + `/health`, локально `npm ci && scripts/ci-push.sh` зелёный.
- `main` защищён на GitHub; работа только через feature-ветки + PR.
- Проходит хотя бы один автономный прогон бота: задача → ветка → зелёные гейты.

**Метрики «получилось» (для полигона):**
- бот добавляет крошечную фичу (напр. новый эндпоинт + тест) без вмешательства, гейты зелёные;
- ground-truth = результат гейтов/CI, а не самооценка бота;
- нет прямых push в `main` (branch protection работает).
