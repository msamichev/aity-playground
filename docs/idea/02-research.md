## Содержание

Todo-list REST API — канонический «hello world» бэкенда: существует масса референсных
реализаций (TodoMVC, REST-туториалы на Express/Fastify). Это **намеренный** выбор:
стандартность означает предсказуемость — легко писать тесты и проверять корректность бота.

Более широкий контекст — в соседнем инструменте agent-aity (исследование по изоляции в
Docker, авторизации по API-ключу дешёвой модели DeepSeek и сторожу против runaway-прогонов).
Полигон существует, чтобы проверить эти выводы на практике, не рискуя реальным проектом.
