# 01. Idea

> **Frozen snapshot.** Заполняется командой `/init-project` из inbox/ и обсуждений.
> После заполнения — не редактируем спонтанно. Изменения — через ADR.

> Сердцевина: одно-два предложения «что это» и «зачем».

## Содержание

**Что:** маленький REST-сервис списка задач (todo). Backend-only, без UI и без БД —
хранилище **в памяти** (рестарт = чистое состояние).

**Зачем:** тренировочный полигон для отладки механики контейнерных ботов «aity»
(инструмент `agent-aity`). Сервис — нагрузка, а не самоцель.

**Принятые идейные решения:**
- in-memory вместо БД — осознанно, ради простоты и скорости тестов;
- стандартный JSON REST без аутентификации — чтобы у каждой фичи был однозначный ожидаемый ответ;
- скоуп заморожен; рост — только узкими задачами от ботов;
- git-хост — GitHub (не GitLab, как обычно в team+ai); `main` защищён, работа через PR.
